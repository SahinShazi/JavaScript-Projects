# Password Validator

A real-time password strength validator with visual feedback.

## Features

- Real-time password validation
- Show/hide password toggle
- Visual checklist for password requirements
- Interactive feedback as you type

## Password Requirements

- Minimum 8 characters
- At least one number (0-9)
- At least one lowercase letter (a-z)
- At least one uppercase letter (A-Z)
- At least one special character

## How It Works

The validator checks your password in real-time and displays which requirements are met with a visual checklist. Click the eye icon to toggle password visibility.
